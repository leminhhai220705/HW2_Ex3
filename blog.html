TodoList app

The app will allow users to:

  • Add new tasks
  • Display the list of tasks
  • Delete tasks

1. App Overview:

We’re going to build a simple Todo List application that includes:

  • An input box to type in a task
  • A button to add the task
  • A list that shows all tasks
  • A delete button to remove a task

2. Create the HTML Structure

First, create a file called index.html:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Todo List</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <div class="todo-container">
    <h1>Todo List</h1>
    <div class="input-section">
        <input type="text" id="todo-input" placeholder="Enter a task...">
        <button id="add-btn">Add</button>
    </div>
    <ul id="todo-list"></ul>
    </div>
    <script src="script.js"></script>
</body>
</html>

Explanation:

  • The input (#todo-input) lets the user type in a task.
  • The button (#add-btn) is used to add the task to the list.
  • The list (#todo-list) displays all added tasks.
  • The JavaScript will be added in a separate file called script.js.

4. Add Functionality with JavaScript

Create a file called script.js:

const addBtn = document.getElementById('add-btn');
const input = document.getElementById('todo-input');
const list = document.getElementById('todo-list');

addBtn.addEventListener('click', function () {
    const task = input.value.trim();
    if (task !== '') {
    const li = document.createElement('li');
    li.textContent = task;

    const deleteBtn = document.createElement('button');
    deleteBtn.textContent = 'Delete';
    deleteBtn.className = 'delete-btn';
    deleteBtn.onclick = function () {
        list.removeChild(li);
    };

    li.appendChild(deleteBtn);
    list.appendChild(li);

    input.value = ''; // clear input after adding
    }
});

Explanation:

  • We get references to the input field, the add button, and the list.
  • When the Add button is clicked:
    • We check if the input is not empty.
    • A new <li> is created for the task.
    • A Delete button is added inside the <li>
    • When the Delete button is clicked, it removes the task from the list.

Final Result

Now, when you open your index.html in a browser:

  • Type a task in the input field and click Add.
  • The task will appear in the list with a Delete button.
  • Clicking Delete will remove that task.
